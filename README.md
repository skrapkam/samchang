## Development

Clone the repository:
`git clone https://github.com/skrapkam/samchang.git`

Install [Gatsby](https://www.gatsbyjs.org/): `npm i -g gatsby-cli`

`cd` into the directory:
`cd samchang`

Run `npm install` or `yarn` to install dependencies

Run `gatsby develop`

Open the site:
`localhost:8000`
